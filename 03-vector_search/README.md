## 💡 벡터 검색의 강점

### 키워드 검색 vs 벡터 검색 비교

| 검색 쿼리 | 키워드 검색 | 벡터 검색 |
|----------|------------|----------|
| "가벼운 외출복" | ❌ 결과 없음 (단어 불일치) | ✅ 바람막이, 가디건 검색됨 |
| "선물하기 좋은 제품" | ❌ 결과 없음 | ✅ 기프트 세트, 선물 포장 상품 검색됨 |
| "운동할 때 입기 좋은 옷" | ❌ 결과 없음 | ✅ 트레이닝복, 레깅스 검색됨 |

### 벡터 검색이 필요한 상황

- 🔤 **동의어/유의어 검색**: "노트북" → "랩탑", "휴대용 컴퓨터"
- 🌐 **자연어 질문**: "여름에 시원하게 입을 수 있는 옷"
- 💭 **추상적 개념 검색**: "고급스러운 느낌의 가방"

## 📊 HNSW vs Exhaustive KNN

| 구분 | HNSW | Exhaustive KNN |
|------|------|----------------|
| **속도** | ⚡ 빠름 (근사치) | 🐢 느림 (정확도 우선) |
| **정확도** | 약 95~99% | 100% |
| **메모리** | 추가 인덱스 필요 | 인덱스 없음 |
| **사용 케이스** | 대용량 데이터, 실시간 검색 | 소규모 데이터, 정확도 중요 |
