## 🚀 사용 방법

1. `sample_products.csv` 파일을 로드합니다
2. Azure AI Search 인덱스에 데이터를 업로드합니다
3. 이미지 URL을 통해 벡터 임베딩을 생성합니다
4. 검색 쿼리를 실행합니다

```python
import pandas as pd

# 데이터 로드
df = pd.read_csv('sample_data.csv')

# 데이터 확인
print(df.head())
print(f"총 상품 수: {len(df)}")
```
