# 7️⃣ 생성형 AI를 활용한 인덱스 데이터 증강

이 모듈에서는 **GPT** 모델의 Vision 기능을 사용하여 제품 이미지를 분석하고, 분석 결과를 인덱스 데이터에 추가하여 **검색 품질을 향상**시키는 방법을 학습합니다.

## 📋 학습 목표

- GPT 모델을 사용한 이미지 분석
- 인덱스 데이터 증강(Enrichment) 개념 이해
- 증강된 데이터로 벡터 재생성 및 인덱스 업데이트
- 검색 품질 향상 검증

## 🎯 데이터 증강이 필요한 이유

### 기존 인덱스의 한계

기존 `content_vector`는 다음 정보만 포함:
- 제품명 (title)
- 브랜드 (brand)
- 카테고리 (category)

```
**제품명**: [어그] 여성 플러프 예 슬라이드 플랫폼 샌들 블랙
**브랜드**: 어그
**카테고리**: 패션잡화
```

### 검색 한계 예시

| 검색어 | 결과 | 문제점 |
|--------|------|--------|
| "따뜻한 겨울 신발" | ❌ 검색 안됨 | "따뜻한", "겨울" 정보 없음 |
| "부드러운 소재" | ❌ 검색 안됨 | 소재/재질 정보 없음 |
| "편안한 착용감" | ❌ 검색 안됨 | 착용감 정보 없음 |

### 이미지 분석으로 추출 가능한 정보

LLM이 이미지에서 추출할 수 있는 정보:

1. **색상 및 디자인**: 검정, 모던한 디자인, 심플한 등
2. **소재/재질**: 푹신한 퍼, 부드러운 소재, 고급 가죽 등
3. **사용 시나리오**: 실내용, 휴양지, 캐주얼 등
4. **타겟 고객**: 20-30대 여성, 편안함 추구 고객 등
5. **마케팅 포인트**: 고급스러운, 트렌디한, 실용적인 등

## 🔄 데이터 증강 프로세스

```
┌─────────────────────────────────────────────────────────┐
│                    1. 데이터 로드                        │
│         CSV 파일에서 제품 정보 및 이미지 URL 로드          │
└───────────────────────────┬─────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                   2. 이미지 분석                         │
│    GPT 모델로 제품 이미지 → 텍스트 특징 추출       │
│                                                          │
│    입력: 이미지 URL                                       │
│    출력: "푹신한 퍼 소재의 편안한 슬라이드 샌들로..."      │
└───────────────────────────┬─────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                   3. 콘텐츠 증강                         │
│          기존 텍스트 + 이미지 분석 결과 결합              │
│                                                          │
│    **제품명**: [title]                                    │
│    **브랜드**: [brand]                                    │
│    **카테고리**: [category]                               │
│    **제품 특징**: [AI 분석 결과]   ← 새로 추가!           │
└───────────────────────────┬─────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                   4. 임베딩 재생성                        │
│        증강된 콘텐츠를 벡터로 변환 (1536차원)             │
└───────────────────────────┬─────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                  5. 인덱스 업데이트                       │
│          content_vector 필드에 새 벡터 업로드             │
└─────────────────────────────────────────────────────────┘
```

## 📊 증강 전 vs 증강 후 비교

### content_text 비교

| 구분 | 증강 전 | 증강 후 |
|------|--------|---------|
| 제품명 | ✅ | ✅ |
| 브랜드 | ✅ | ✅ |
| 카테고리 | ✅ | ✅ |
| 제품 특징 | ❌ | ✅ (AI 생성) |
| 색상/디자인 | ❌ | ✅ |
| 소재/재질 | ❌ | ✅ |
| 사용 시나리오 | ❌ | ✅ |

## 💡 GPT Vision 프롬프트 설계

### System Prompt

```
당신은 전문 상품 마케터이자 쇼핑몰 판매 전문가입니다.
제품 이미지를 보고 고객에게 어필할 수 있는 핵심 특징을 분석해주세요.

다음 관점에서 분석하세요:
1. 색상 및 디자인 특징
2. 소재/재질 (이미지에서 유추 가능한 경우)
3. 사용 시나리오/용도
4. 타겟 고객층
5. 마케팅 어필 포인트
```

### 결과 예시

**입력 이미지**: 어그 슬라이드 샌들
**분석 결과**:
> "블랙 컬러의 푹신한 퍼 슬라이드 샌들입니다. 두꺼운 플랫폼 디자인이 트렌디하며, 
> 실내외 겸용으로 활용도가 높습니다. 20-30대 여성을 타겟으로 하며, 
> 편안함과 스타일을 동시에 추구하는 고객에게 적합합니다."
